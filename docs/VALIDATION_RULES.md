# Validation Rules المختصرة

- لا يمكن تأجير سيارة ليست Available.
- لا يمكن تأجير سيارة في الورشة أو حادث أو مباعة.
- رقم اللوحة ورقم الهيكل فريدان.
- قراءة العداد لا تنقص عن القراءة السابقة.
- لا يمكن إغلاق أمر صيانة بدون ملاحظات إغلاق.
- لا يمكن تسجيل مبالغ سالبة.
- وعد السداد يتحول إلى Promise Broken عند تجاوزه بدون سداد.
- جدول أقساط التمويل يولد عند Submit عقد التمويل.
- لا يمكن ربط قضية دون عميل أو عقد أو تحصيل أو مطالبة.
